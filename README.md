# 🍽️ Hotel Yashdeep

A modern full-stack Restaurant Management System built with the MERN Stack.

Hotel Yashdeep is designed to digitize restaurant operations by providing an efficient admin dashboard for managing reservations, tables, menu items, customers, and restaurant content while delivering a seamless customer experience.

---

### 👨‍💼 Admin Dashboard
- Dashboard Overview
- Reservation Management
- Table Management

- Responsive Admin Interface

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Hotel-Yashdeep.git
cd Hotel-Yashdeep
```

---

### Install Dependencies

Frontend

```bash
cd frontend
npm install
```

Backend

```bash
cd backend
npm install
```

---

### Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

NODE_ENV=development
```

---

### Run Backend

```bash
cd backend
npm run dev
```

---

### Run Frontend

```bash
cd frontend
npm run dev
```
